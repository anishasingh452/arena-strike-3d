# Arena Strike 3D

Arena Strike 3D is a browser-based first-person shooter built using Three.js. Players battle against an AI-controlled opponent in a 3D arena using simple keyboard controls.

## Features

- 3D arena environment with walls, cover pillars, lighting, and grid floor
- First-person shooter gameplay
- AI enemy with patrol, chase, and attack behaviors
- Health bars for both player and enemy
- Score tracking system
- Win/Lose game states with replay functionality
- Lightweight and runs directly in the browser

## Controls

| Key | Action |
|------|---------|
| ↑ | Move Forward |
| ↓ | Move Backward |
| ← | Turn Left |
| → | Turn Right |
| Space | Shoot |

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
